# 📊 Sales Performance Dashboard

## Project Overview

This project analyzes a sales dataset using Python. It includes data cleaning, KPI calculation, and simple visualizations to understand business performance.

## Tools Used

- Python
- Pandas
- Matplotlib

## Project Steps

- Load Dataset
- Clean Data
- Create KPIs
- Visualize Data
- Business Insights

## KPIs

- Total Sales
- Total Orders
- Total Profit
- Top Products
- Top Customers

## Output

- Cleaned Dataset
- Charts
- Business Insights
